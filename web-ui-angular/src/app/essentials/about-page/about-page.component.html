About Posther

Posther is a proof of concept dApp deployed on Ethereum Kovan. It was created with two major goals in mind.

It is an open source project to showcase some basics of blockchain development on all levels of the stack. You can check out the deployed contract on Etherscan. The complete source code is available on Github.

It aims to enable the persistence of plain text on Ethereum with the lowest possible gas fees .

Posted text is stored as calldata. No on-chain events are emitted, no EVM storage is used.

The stored text can only be retrieved by scanning transactions related to the contract, but the contract state itself holds no such information.
